# Job-Lander
A full stack JavaScript application for recent graduates who want to organize their job hunt

 # Live Demo
 https://job-lander.herokuapp.com
 
# Technologies
<ul>
<li>React.js</li>
<li>Node.js</li>
<li>Express.js</li>
<li>JSON.server</li>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
<li>Material UI</li>
<li>AWS</li>
<li>Multer</li>
<li>Multer S3</li>
  </ul>
